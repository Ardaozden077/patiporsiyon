# Patiporsiyon

Patiporsiyon, köpekler için kişiselleştirilmiş mama planı oluşturan modern bir web uygulamasıdır. Kullanıcı; köpeğinin yaşı, kilosu, aktivite seviyesi, hedefi ve tarif tercihine göre günlük porsiyon, plan özeti ve tahmini fiyat bilgisi alır.

## Özellikler

- Kişiselleştirilmiş mama planı oluşturma
- Günlük gramaj ve plan özeti hesaplama
- Tarif ve hedef bazlı fiyatlandırma
- Demo giriş / kullanıcı paneli akışı
- Admin paneli arayüzü
- Modern ve premium odaklı UI

## Teknolojiler

- React
- Vite
- JavaScript
- CSS
